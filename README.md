
# Literal Launcher (L²)

**Literal Launcher (L²)** is an ultra-lightweight text-based launcher for Android, designed to eliminate visual noise. True to its name "L²" (L squared), it aims to deepen the philosophy of minimalism.

## Overview
Icons, colors, and widgets have been discarded. The UI consists purely of text on a black background, minimizing distractions and maximizing user focus.

## Inspiration
L² inherits the spirit of **μLauncher (mu-launcher)**, the pioneer of minimalist launchers. While honoring its simplicity, L² was developed to incorporate personal preferences for a more refined daily experience.

## Unique Feature: The Chest
A dedicated storage for "second-tier" apps.
* **Organization**: Keep only your essential "first-tier" apps on the home screen; send everything else to the Chest.
* **Utility**: Launch apps directly from the Chest list without cluttering your home screen.
* **Restore**: Tap the icon on the right side of a Chest item to return the app to the main drawer.

## Layout (Invisible Slots / UI)
The home screen is built on a 3x3 invisible grid. The left and right columns provide six app slots, while the center column is dedicated to information and system tasks.

| Left | Center | Right |
| :--- | :--- | :--- |
| **Slot: Top Left** | (Empty / Swipe Up Area) | **Slot: Top Right** |
| **Slot: Mid Left** | **Clock / Date / Battery** | **Slot: Mid Right** |
| **Slot: Bot Left** | **Notification (Tap)** | **Slot: Bot Right** |

* **Display Settings**: Clock, Date, and Battery visibility can be toggled individually in settings.
* **Bottom Center (Notification)**: Tap to expand the notification panel.
* **Center (Info)**: Tap each info element to launch its assigned app.

## Controls

### Home Screen
* **Swipe Up**: Open App Drawer.
* **Double Tap**: Launch the assigned "Shortcut App".
* **Tap Slot**: Launch assigned app.
* **Long Press Slot**: Open App Picker to assign a slot.

### App Drawer
* **Tap**: Launch app (Drawer closes automatically).
* **Tap Background (Empty Space)**: **Return to Home** (No back gesture required).
* **Long Press**: Show menu (Rename / Send to Chest / App Info).
* **Scroll**: Dedicated vertical scroll (No accidental "back to home" triggers during scrolling).
* **"chest" (Bottom Right)**: Open Chest / Settings (Toggle Info, Scale, Font, Slot assignment).

**Literal Launcher (L²)** は、視覚的ノイズを排除した Android 用のテキストランチャー。

## 概要
アイコン、色、ウィジェットを廃し、背景は黒、表示は文字のみに限定。ユーザーの集中力を最大化することを目的とする。

## インスパイア
ミニマルランチャーの先駆である **μLauncher (mu-launcher)** の思想を継承。そのシンプルさをベースに、独自の希望を形にするために開発。

## 独自機能：Chest（チェスト）
二軍アプリの保管庫。
* **整理**: ホームには常用するアプリのみを表示し、それ以外を Chest へ送る。
* **利便性**: Chest 内の一覧から直接アプリを起動可能。
* **復帰**: Chest 内の一覧にある右側のアイコンをタップで、メインのアプリ一覧へ戻す。

## 画面構成（Invisible Slots / UI）
ホーム画面は 3x3 のグリッドで構成。左右の列に計6つのアプリスロットを配置。中央列は情報表示とシステム操作。

| 左 (Left) | 中央 (Center) | 右 (Right) |
| :--- | :--- | :--- |
| **Slot: Top Left** | (空き / スワイプ開始エリア) | **Slot: Top Right** |
| **Slot: Mid Left** | **Clock / Date / Battery** | **Slot: Mid Right** |
| **Slot: Bot Left** | **Notification (Tap)** | **Slot: Bot Right** |

* **表示設定**: 時計・日付・バッテリー残量は、設定により表示/非表示を個別に選択可能。
* **中央下 (Notification)**: タップで通知パネルを展開。
* **中央 (Info)**: 各情報のタップで割り当てアプリを起動。

## 操作一覧

### ホーム画面
* **上スワイプ**: アプリドロワーを開く
* **ダブルタップ**: 設定した「Shortcut App」を起動
* **各スロットをタップ**: 割り当てたアプリを起動
* **各スロットを長押し**: アプリの割り当て設定（Picker）

### アプリドロワー
* **タップ**: アプリを起動（起動後、自動でドロワーを閉じる）
* **背景（余白）をタップ**: ホームに戻る
* **長押し**: メニュー表示（Rename / Send to Chest / App Info）
* **スクロール**: 縦スクロール専用（誤操作によるホーム戻りなし）
* **右下 "chest"**: Chest / 各種設定（表示ON/OFF・スケール・フォント・スロット）を開く
